# ROV Mission Intelligence Platform (RMIP)

## 🚀 Live Demo
https://rmip-app-otd.streamlit.app/

## 📊 Overview
The ROV Mission Intelligence Platform (RMIP) is an interactive dashboard designed to analyze subsea telemetry data from ROV operations.

It enables engineers and offshore professionals to visualize mission performance, detect anomalies, and gain operational insights from real-world datasets.

---

## ⚙️ Features

- 📂 Upload and analyze mission CSV logs
- 🔄 Automatic handling of inconsistent CSV formats (`,` `;` etc)
- 🧠 Smart column detection (timestamp, depth, etc)
- 📈 Depth vs Time visualization
- 📊 Key performance indicators (Max Depth, Mean Depth)
- ⚠️ Operational alerts (signal loss, depth limits)
- 🌍 3D mission visualization (latitude, longitude, depth)
- 📍 Mission coverage analysis

---

## 🧪 Use Case

This tool simulates real-world subsea operations monitoring, where data often comes in inconsistent formats and requires preprocessing before analysis.

It is particularly relevant for:
- ROV operations
- Subsea inspections
- Offshore data analysis
- Engineering decision support
